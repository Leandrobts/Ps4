Estas primitivas combinadas representam um risco de segurança severo, fornecendo os blocos de construção fundamentais para exploits mais complexos, incluindo potencial RCE. **Vulnerabilidades Detalhadas:** 1. **CSP Bypass / Cross-Site Scripting (XSS):** * **Descrição:** O navegador permite a execução de JavaScript via URIs `data:` e manipuladores `onerror`, indicando uma falha na aplicação da CSP. * **Impacto:** Permite a execução inicial de código arbitrário no contexto da página. * **Demonstração:** O Teste 1 no script PoC aciona `alert()`s e modifica o DOM. Permite ler memória adjacente que pode conter ponteiros internos (para código ou heap), metadados ou outros dados sensíveis.
